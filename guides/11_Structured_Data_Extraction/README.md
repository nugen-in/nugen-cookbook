# Structured Data Extraction with Nugen

## 📋 Problem Statement
In many workflows, data exists in unstructured formats like emails, resumes, invoices, or log files. Extracting this data into a structured format (JSON, CSV, Database) is time-consuming and error-prone when done manually or with rigid value extraction rules (regex).

## 🎯 Target Audience
- Developers building data processing pipelines.
- Data Analysts converting unstructured text to structured datasets.
- AI Engineers looking to integrate Nugen into their ETL workflows.

## ✅ Prerequisites
- A Nugen API Key (Sign up at [nugen.in](https://nugen.in)).
- Python 3.8 or higher.
- Basic understanding of REST APIs.

## 🚀 Expected Outcomes
By the end of this cookbook, you will:
1.  Understand how to prompt Nugen models for valid JSON output.
2.  Build a reusable function to extract specific fields from text.
3.  Handle errors and validate the extracted data.

## 🛠️ Setup
1.  Copy `.env.example` to `.env` and add your `NUGEN_API_KEY`.
2.  Install dependencies: `pip install -r requirements.txt`.
3.  Run the notebook!
